List comprehension adalah cara ringkas untuk membuat list baru dari iterable yang ada. Sintaksnya:

[ekspresi for item in iterable if kondisi]
Contoh sederhana — membuat list bilangan kuadrat: 
# Cara biasa
squares = []
for x in range(10):
    squares.append(x**2)

# Dengan list comprehension ✨
squares = [x**2 for x in range(10)]
# Output: [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]
Bisa juga ditambah kondisi filter, misalnya hanya bilangan genap: 
evens = [x for x in range(20) if x % 2 == 0]
Mau saya jelaskan lebih lanjut, seperti nested list comprehension?
